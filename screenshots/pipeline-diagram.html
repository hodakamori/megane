Node Types

Data Loader

Loads structure files (.pdb, .gro, .xyz, .mol, .sdf)

Out: particlebondcell

Filter

Query-based filtering. Accepts particle or bond.

In: particle|bond
Out: same type

Modify

Visual property modification (scale, opacity)

In: particle|bond
Out: same type

Label Generator

Generates labels from particle data

In: particle
Out: label

Viewport

Terminal sink. Renders all connected data.

In: particlebondcelllabelmesh
Out: none
